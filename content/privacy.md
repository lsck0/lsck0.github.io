# Privacy Policy

## 1. Overview

This website is served as a static site via GitHub Pages. No personal data is collected, stored, or processed by the operator.

## 2. Hosting

This website is hosted on GitHub Pages. The hosting provider may collect server log files that document your visit. Details can be found in the [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

## 3. External Resources

This website loads fonts and JavaScript libraries from CDN services (jsDelivr, tikzjax.com). When fetching these resources, your IP address is transmitted to the respective providers.

## 4. Local Storage

This website stores the following data in your browser's localStorage:

- **Theme preference** (light/dark mode)
- **Reading history** (which posts you have read)
- **Bookmarks** (posts you have bookmarked)

This data does not leave your browser and is never transmitted to any server.

## 5. Comments

Comments are provided by [Giscus](https://giscus.app/), which uses GitHub Discussions. When you interact with the comment section, data is processed according to [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

## 6. AI Scraping

This website explicitly opts out of AI model training. See the [Terms of Service](/tos) for details.

## 7. Your Rights

For privacy inquiries, contact the site owner via the methods listed on the [about page](/about).
